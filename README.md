# medipus-web

Medipus webapp

## Dev requirements

npm: <https://www.npmjs.com/get-npm>

## Setup local environment

```git clone https://github.com/LSD2evosoft/medipus-web```

```cd medipus-web```

```npm install```

## Run locally

```ng serve```

Open <http://localhost:4200/> in browser.
